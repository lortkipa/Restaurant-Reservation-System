EmailJS Settings

How to get your EmailJS keys
Setup Guide
Go to emailjs.com and create a free account. Free tier allows 200 emails/month.
In the dashboard, go to Email Services → Add a new service (Gmail, Outlook, etc.). Copy the Service ID.
Go to Email Templates → Create New Template. Design your email using the template variables listed below. Copy the Template ID.
Go to Account → General. Copy your Public Key.
Paste all three keys into the form below and click Save Changes. Use the Test panel to verify everything works.
